Andreea's PerfumStore

Bine ai venit la Andreea's PerfumStore! Magazin online de parfumuri dezvoltat în Django Python, cu o experiență captivantă pentru cumpărături.

🛍️ Funcționalități Utilizator:

  • Vizualizare produse în carduri responsive
  • Detalii individuale despre produse
  • Adăugare produse în coș, checkout complet
  • Autentificare și dashboard personal
  • Urmărire comenzi

⚙️ Funcționalități Admin:

  • Adăugare, editare și ștergere produse
  • Tabel cu lista produselor

🛠️ Tehnologii:

  • Django Framework, Python
  • HTML, CSS, JavaScript
  • Django Templates, Authentication, Sessions
